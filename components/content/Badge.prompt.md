Small uppercase category tag / status pill. Use to label blog posts (Travel, Trek, Recipe, Story) and to flag states.

```jsx
<Badge tone="trek">Trek</Badge>
<Badge tone="recipe" solid>Gluten-free</Badge>
<Badge tone="travel" size="sm">Travel</Badge>
```

Tones map to taxonomy colors (`travel` spice, `trek` pine, `recipe` turmeric, `story` sky) plus `neutral / success / warning / danger`. `solid` fills the pill; default is a soft tint.
